# SPECIFICATION

## 1. 概要
Three.js (WebGL) を用いた、アンティーク・クラシックな洋書風デザインの3D本オープニングアニメーションと、インタラクティブな本棚（Bookshelf）のフロントエンド実装。
本をクリックすると中央で開き、その見開きページ内にHTML/CSSによるリッチな動的コンテンツ（パートナーロゴ、アンケートフォーム、コメント掲示板、ソーシャルアカウントリンク）が表示され、右上から「戻る」ボタンで本棚へシームレスに戻れるSPA（シングルページアプリケーション）構成のポートフォリオ。

## 2. UX仕様（演出フロー）
### 2.1. オープニングシーン
* **初期状態**: 空間中央に閉じたアンティーク調の3Dの本が配置されている。
  * タイトルはゴールド箔押しで **「TEDxUTsukuba Salon \n Library」** と刻印されている。
* **アニメーション**:
  1. ページロード時、自動的に本がカメラに向かって滑らかに近づいてくる（Z軸移動）。
  2. 同時に、表紙が左側を軸に180度滑らかに開ききる。
  3. カメラが本の中身に向かって急接近し、ページの中に飛び込むようなズームイン効果。
  4. 画面全体がホワイトアウトする。
  5. オープニングシーンからメインの本棚シーンへ移行。

### 2.2. メインシーン（本棚）
* **本棚構造**: 木製の本棚に4冊の洋書が背表紙を向けて隙間なく並んで格納されている。
  * カメラは少し引いた位置から本棚全体を見渡す。
  * 各背表紙の文字（ゴールド箔押し、黒縁トローク、影）で高い可読性を維持。
  * 本のデータ構成:
    1. **パートナー (Partners)**: `title: "パートナー\nOur Partners"`, `spineTitle: "Partners"`, `url: "#partners"`
    2. **アンケート (Survey)**: `title: "アンケート\nQuestionnaire"`, `spineTitle: "Survey"`, `url: "#survey"`
    3. **コメント (Comments)**: `title: "Comment\nScreen"`, `spineTitle: "Comments"`, `url: "#comments"`
    4. **アカウント (Socials)**: `title: "アカウント\nSocial Accounts"`, `spineTitle: "Socials"`, `url: "#socials"`

### 2.3. インタラクション (クリック展開と閉じる挙動)
* **クリック展開**:
  1. 本を選択しクリックすると、その本が画面中央へ引き出され、カメラがズームインする。
  2. 表紙が180度全開になり、本の内側から光が漏れる。
  3. **本が開ききった瞬間、その本の手前に「見開きページ」を模したHTMLコンテンツオーバーレイ（`#content-overlay`）がフェードインする。**
  4. 同時に、画面右上に**アンティークゴールド調の「戻る（Close / ✕）」ボタン**がフェードインする。
* **コンテンツ表示**:
  * 開いた本の種類に応じて、以下の見開きコンテンツを表示する：
    * **パートナー**: スタイリッシュに配置されたパートナー企業ロゴ一覧（SVG/CSSによる架空のロゴグリッド）。
    * **アンケート**: 美しくスタイリングされたフォーム（お名前、メール、送信ボタン）。
    * **コメント**: メモ用紙風のコメント閲覧＆入力掲示板。
    * **アカウント**: X (Twitter), Instagram, GitHub などの美麗なソーシャルアカウントロゴリンク。
* **戻る（閉じる）挙動**:
  1. 画面右上の「戻る」ボタンをクリックすると、`#content-overlay` と「戻る」ボタンがフェードアウト。
  2. 対象の本の表紙がパタッと閉じる（`hinge.rotation.y` が `0` に戻る）。
  3. 本が画面中央から、元の本棚のスロットに滑らかに戻る。
  4. 奥に退避していた他の本たちが元の並びに復帰し、透明度が `1.0` に戻る。
  5. カメラの位置・アングルが本棚全体を見渡す引きの位置に戻る。
  6. トランジションが完了し、ユーザーは再び別の本を選択可能な状態に戻る（シームレスなループUX）。

## 3. 技術仕様
* **言語/ライブラリ**: HTML5, CSS3, JavaScript, Three.js (CDN), GSAP (CDN)
* **読みにくさ解消のCanvasテキスト描画**:
  * 背表紙・表紙の金箔文字に `strokeStyle = '#000000'`, `lineWidth = 6` による黒フチを追加。
  * `shadowColor = 'rgba(0, 0, 0, 0.95)'`, `shadowBlur = 12` による豊かな陰影を追加。
  * 文字のバラバラ化（`split("").join(" ")`）を廃止し、まとまった単語として描画。
* **見開きコンテンツオーバーレイ (`#content-overlay`)**:
  * アスペクト比を維持しつつ、PC/スマホでレスポンシブなサイズ（`width: 90%; max-width: 800px; height: 80vh; max-height: 550px; aspect-ratio: 4.4 / 3.2;`）のアイボリー調の紙テクスチャ背景を持つHTMLコンテナ。
  * 左右2カラム（見開きの左ページ・右ページに対応）に分割し、フォーム入力やリンクターゲットのクリックに対応。
* **閉じるボタン (`#close-button`)**:
  * `pointer-events: auto` で画面右上に絶対配置。ホバー時にゴールドの輝き（ボカシ効果）を適用するアンティーク調ボタン。
* **GSAP 逆再生アニメーション**:
  * タイムラインまたは個別のイージングアニメーションで、本の閉鎖・本棚への格納・他書の復帰・カメラの後退を完璧に同期させて逆再生。
* **コード構造**: 全てを単一のファイル（`index.html`）に内包。
