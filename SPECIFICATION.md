# SPECIFICATION

## 1. 概要
Three.js (WebGL) を用いた、アンティーク・クラシックな洋書風デザインの3D本オープニングアニメーションと、インタラクティブな本棚（Bookshelf）のフロントエンド実装。スマートフォン（縦画面）対応、高解像度テキスト表現、およびドラマチックな飛び込みカメラワークを適用したバージョン。

## 2. UX仕様（演出フロー）
### 2.1. オープニングシーン
* **初期状態**: 空間中央に閉じたアンティーク調の3Dの本が配置されている。
  * タイトルはゴールド箔押しで **「TEDxUTsukuba Salon \n Library」**（改行表示）と刻印されている。
* **アニメーション（ページ飛び込みカメラワーク）**:
  1. ページロード時、自動的に本がカメラに向かって滑らかに近づいてくる（Z軸移動）。
  2. 同時に、表紙が左側を軸に180度滑らかに開ききる（見開き状態になる）。
  3. 表紙が開ききるとほぼ同時に、**カメラが本の中身（真っ白なページ）に向かって一気に急接近し、ページの中に飛び込むようなズームイン効果**を発生させる。
  4. 飛び込む瞬間に合わせて、本の内側のポイントライトが最大発光し、画面全体が滑らかに「真っ白（`#ffffff`）」にホワイトアウト（フェードアウト）する。
  5. オープニングシーンのキャンバスを非表示にし、メインの本棚シーンへ移行。

### 2.2. メインシーン（本棚）
* **フェードイン**: ホワイトアウト状態から、重厚な木製の本棚がフェードインする（3D空間）。
* **背景デザイン**: 暖かみのある間接照明に照らされた、ダークでアンティークな空間。
* **本棚構造**: 3Dで表現されたコンパクトな木製の本棚（幅 `5.0`）に、4冊の洋書が背表紙を向けて隙間なく並んで格納されている。
  * ゆったりと本棚全体を見渡せるよう、**カメラの初期位置を従来より少し後退させ（少し引いた位置から見る）**、落ち着いたアングルを形成する。

### 2.3. インタラクション
* **ホバー（Hover / タッチ）**:
  * マウスカーソルを合わせる（またはスマホでタップする）と、その本が手前（Z軸）にスッと引き出され、上に少し持ち上がる。
* **クリック（Click / タップ）**:
  * 本をクリックすると、手前中央に引き出される。
  * カメラの正面で本が回転し、**表紙が完全に開ききる（180度見開き状態になる）**。
  * 開ききると同時に光が漏れ、画面全体が暗転（ブラックアウト）し、ハッシュ遷移（例: `#profile`）する。

## 3. 技術仕様
* **言語/ライブラリ**: HTML5, CSS3, JavaScript, Three.js (CDN), GSAP (CDN)
* **改行対応の Canvas 箔押しテクスチャ**:
  * タイトル文字列に `\n` または `\\n` が含まれている場合、テキストを分割して自動的に複数行に渡り適切な行間（Y座標シフト）でゴールド箔押しテキストをレンダリングする描画ロジックを実装。
  * 複数行の際はフォントサイズを自動で少し縮小（例: `54px`）し、枠内に綺麗に収める。
* **飛び込みカメラワーク (GSAP)**:
  * 本のZ位置 `2.2` に対して、カメラを最終的に `Z = 2.35`, `Y = -0.15` 付近まで一気にズームインさせるイージングアニメーション（`ease: "power2.in"`）を適用し、紙面を突き抜けるような錯覚を作る。
* **本棚アングルの調整（カメラ後退）**:
  * カメラの基本距離を従来のPC `3.5` から **`4.5`** に変更。
  * スマホ（縦画面）時の距離も従来の `4.2 + (1.0 - aspect) * 2.0` から **`5.2 + (1.0 - aspect) * 2.2`** に後退させ、余裕を持って本棚全体が収まるようにする。
* **クリック時の開ききり演出**:
  * クリックされた本の表紙（ヒンジ）の回転量を `-Math.PI * 0.35` (約60度) から **`-Math.PI * 0.95`** (ほぼ180度) に拡大。
* **コード構造**:
  * HTML、CSS、JavaScriptを単一のファイル（`index.html`）に内包。
