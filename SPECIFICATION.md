# SPECIFICATION

## 1. 概要
Three.js (WebGL) を用いた、アンティーク・クラシックな洋書風デザイン of 3D本オープニングアニメーションと、インタラクティブな本棚（Bookshelf）のフロントエンド実装。
本をクリックすると中央で開き、見開きページ内にHTML/CSSで動的コンテンツ（ロゴ、フォーム、掲示板、SNSリンク）を表示し、「戻る」ボタンで本棚へシームレスに戻るSPA構成。背表紙の文字を正対させた縦積み型（Stacked Vertical）にして可読性を追求したバージョン。

## 2. UX仕様（演出フロー）
### 2.1. オープニングシーン
* **初期状態**: 空間中央に閉じたアンティーク調の3Dの本が配置されている。
  * タイトルはゴールド箔押しで **「TEDxUTsukuba Salon \n Library」** と刻印されている。
* **アニメーション**:
  1. ページロード時、自動的に本がカメラに向かって滑らかに近づいてくる（Z軸移動）。
  2. 同時に、表紙が左側を軸に180度滑らかに開ききる。
  3. カメラが本の中身に向かって急接近し、ページの中に飛び込むようなズームイン効果。
  4. 画面全体がホワイトアウトする。
  5. オープニングシーンからメインの本棚シーンへ移行。

### 2.2. メインシーン（本棚）
* **本棚構造**: 木製の本棚に4冊の洋書が背表紙を向けて隙間なく並んで格納されている。
  * カメラは少し引いた位置から本棚全体を見渡す。
  * **背表紙の文字の可読性（視認性）の劇的向上 (Stacked Vertical Layout)**:
    * 文字が90度回転（横倒し）していると頭を傾ける必要があり読みにくいため、**「文字自体は回転させず直立（正対）させたまま、上から下へ縦に1文字ずつ積み重ねて（Stackして）並べる」** 描画方式を採用する。
    * 黒い太めの縁取り（アウトライン）とドロップシャドウの適用を継続し、どのような本の色でも輪郭をくっきりと浮き出させる。
    * 本のデータ構成:
      1. **パートナー**: 表紙 `パートナー\nOur Partners` | 背表紙: `PARTNERS` | URL `#partners`
      2. **アンケート**: 表紙 `アンケート\nQuestionnaire` | 背表紙: `SURVEY` | URL `#survey`
      3. **コメント**: 表紙 `Comment\nScreen` | 背表紙: `COMMENTS` | URL `#comments`
      4. **アカウント**: 表紙 `アカウント\nSocial Accounts` | 背表紙: `SOCIALS` | URL `#socials`

### 2.3. インタラクション (クリック展開と閉じる挙動)
* **クリック展開**:
  1. 本を選択しクリックすると、その本が画面中央へ引き出され、カメラがズームインする。
  2. 表紙が180度全開になり、見開きページを模したHTMLコンテンツオーバーレイ（`#content-overlay`）がフェードイン。
  3. 画面右上にアンティークゴールド調の「戻る（Close / ✕）」ボタンがフェードイン。
* **コンテンツ表示**:
  * 開いた本に応じて、パートナーロゴ、アンケート、コメント掲示板、ソーシャルリンクを表示。
* **戻る（閉じる）挙動**:
  1. 「戻る」ボタンをクリックすると、オーバーレイとボタンがフェードアウト。
  2. 対象の本が閉じて本棚の元のスロットへ滑らかに戻り、他書とカメラの位置も本棚初期アングルに復帰。

## 3. 技術仕様
* **言語/ライブラリ**: HTML5, CSS3, JavaScript, Three.js (CDN), GSAP (CDN)
* **縦積み（Stacked Vertical）Canvasテキスト描画**:
  * `drawGoldFoilDual` 内で `isSpine` が `true` の場合：
    * `rotate(Math.PI / 2)` によるキャンバスの回転を**廃止**。
    * テキストを1文字ずつ分割し、Y方向に行間を加算しながら上から下へ直線的に配置。
    * 縦方向の総高さを計算し、背表紙の中央にバランスよく収まるように描画の開始Y座標（`startY`）を動的計算する。
    * 各文字に強力なシャドウと黒フチ（`strokeText`）を個別に適用し、可読性を最大化する。
* **コード構造**: 全てを単一のファイル（`index.html`）に内包。
