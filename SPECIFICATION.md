# SPECIFICATION

## 1. 概要
Three.js (WebGL) を用いた、アンティーク・クラシックな洋書風デザインの3D本オープニングアニメーションと、インタラクティブな本棚（Bookshelf）のフロントエンド実装。

## 2. UX仕様（演出フロー）
### 2.1. オープニングシーン
* **初期状態**: 重厚な空間の中央に、閉じたアンティーク調の3Dの本（革張りのテクスチャ、金箔押し風の装飾）が配置されている。
* **アニメーション**:
  1. ページロード時、自動的に本がカメラに向かって滑らかに近づいてくる（Z軸移動）。
  2. 同時に、表紙が左側（背表紙をヒンジとする位置）を軸に180度滑らかに開く。
  3. 表紙が開くと、本の内側から眩い白色光が溢れ出す（ポイントライトの強度を急激に上げ、フレア効果のようにカメラを覆う）。
  4. 光が画面全体を覆い尽くし、画面が完全に「真っ白（`#ffffff`）」にフェードアウト（ホワイトアウト）する。
  5. ホワイトアウト完了後、オープニングシーンのThree.jsキャンバスを非表示にし、メインの本棚シーンへ移行する。

### 2.2. メインシーン（本棚）
* **フェードイン**: ホワイトアウト状態から、重厚な木製の本棚がフェードインする（Three.jsで描画された3D空間）。
* **背景デザイン**: 暖かみのあるロウソクや間接照明に照らされたような、ダークでアンティークな空間。
* **本棚構造**: 3Dで表現された木製の本棚に、複数の洋書が背表紙を向けて横並びに並んでいる。
  * 各背表紙には項目名（`Profile`, `Works`, `Contact`, `Blog`）が金箔風のフォントで記載されている。
  * 本のカラーは、クラシックな深みのある色合い（バーガンディレッド、ロイヤルネイビー、フォレストグリーン、アンバーゴールド等）。
  * 革の質感や金箔の文字は、HTML Canvasを用いてJavaScriptで動的に美麗なテクスチャを生成してマテリアルに貼り付ける。

### 2.3. インタラクション
* **ホバー（Hover）**:
  * 背表紙にマウスカーソルを合わせると、その本が本棚からスッと手前（Z軸）に引き出され、上に少し持ち上がる。
  * 本棚の奥にリアルな影が落ち、引き出されることで影の形が変わる。
  * ホバーを外すと滑らかに元の位置に戻る。
* **クリック（Click）**:
  * 本をクリックすると、その本が手前中央に大きく引き出される。
  * カメラの正面で本がくるりと回転し、表紙が少し開くエフェクトが発生する。
  * その後、画面が暗転またはハッシュ遷移（例: `#profile`）する。

## 3. 技術仕様
* **言語/ライブラリ**: HTML5, CSS3, JavaScript (Vanilla JS), Three.js (CDN経由でロード)
* **3D表現 (Three.js)**:
  * `WebGLRenderer`, `PerspectiveCamera`, `Scene` を使用。
  * 本の開閉を制御するため、表紙の回転軸（ヒンジ）を考慮したグループ構成（背表紙＋中身＋表紙の入れ子構造）にする。
* **ライティング**:
  * `AmbientLight`（全体を柔らかく照らす環境光）。
  * `DirectionalLight` または `SpotLight`（本に落ちるリアルな影と、金箔の光沢感を出すためのハイライト）。
  * オープニング本の内側に仕込む `PointLight`（開いた際の眩い光の演出用）。
* **テクスチャ生成**:
  * HTML Canvasを動的に作成し、革のシワのような微細なノイズ、飾り枠（金枠ボーダー）、金箔調のテキストを描画して `CanvasTexture` としてThree.jsのマテリアル（`MeshStandardMaterial`）に適用。
  * `roughness`（粗さ）や `metalness`（金属感）マップを調整し、金箔文字の部分だけキラリと光るようにする。
* **アニメーション**:
  * JavaScript (Three.jsのフレームループ内、または `gsap` などの軽量アニメーションライブラリ、あるいはシンプルなイージング計算）を用いてスムーズな動きを実現。
* **コード構造**:
  * HTML、CSS、JavaScriptを単一のファイル（`index.html`）に内包する。
  * コメントを豊富に記述し、初心者でも3Dの座標調整やテキスト変更ができるようにする。
